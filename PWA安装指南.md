# PWA安装指南 - 最简单的解决方案

## 🚀 为什么选择PWA安装？

### 优势：
- ✅ **无需APK文件**：直接安装到主屏幕
- ✅ **自动更新**：无需手动更新
- ✅ **更稳定**：不会有签名问题
- ✅ **功能完整**：与原生应用体验相同
- ✅ **离线使用**：支持离线功能

### 与APK的对比：
| 特性 | PWA安装 | APK安装 |
|------|---------|---------|
| 安装难度 | ⭐ | ⭐⭐⭐ |
| 稳定性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 更新方式 | 自动 | 手动 |
| 存储空间 | 小 | 大 |
| 权限管理 | 浏览器管理 | 系统管理 |

## 📱 安装步骤

### 方法一：Chrome浏览器（详细步骤）

#### 步骤1：访问网站
1. **打开Chrome浏览器**
2. **在地址栏输入**：
   ```
   https://dpxj.github.io/lan-auto_input_app/
   ```
3. **等待页面完全加载**

#### 步骤2：查找安装按钮
**选项A：地址栏安装**
- 在地址栏右侧查找：
  - 📥 下载图标
  - ➕ 加号图标
  - 📱 手机图标
  - "安装"文字按钮

**选项B：菜单安装**
- 点击右上角：
  - ⋮ 三个点（垂直）
  - ☰ 三个横线
  - ▢ 方块图标
- 在菜单中查找：
  - "添加到主屏幕"
  - "安装应用"
  - "添加到桌面"

#### 步骤3：确认安装
- 点击安装按钮
- 在弹出的对话框中点击"安装"
- 等待安装完成

### 方法二：手动触发安装

如果看不到安装按钮，可以手动触发：

1. **在地址栏输入**：
   ```
   chrome://flags/#bypass-app-banner-engagement-checks
   ```

2. **启用选项**：
   - 找到"Bypass app banner engagement checks"
   - 点击下拉菜单，选择"Enabled"
   - 点击"重启"按钮

3. **重新访问网站**：
   - 重新访问：`https://dpxj.github.io/lan-auto_input_app/`
   - 应该会弹出安装提示

### 方法三：其他浏览器

#### Edge浏览器：
1. 访问网址
2. 点击右上角 ⋯
3. 选择"应用" → "安装此网站作为应用"

#### Firefox：
1. 访问网址
2. 点击右上角 ☰
3. 选择"添加到主屏幕"

#### Samsung Internet：
1. 访问网址
2. 点击右上角 ⋮
3. 选择"添加到主屏幕"

## 🔧 故障排除

### 问题1：看不到安装按钮

**解决方案**：
1. **检查浏览器版本**：
   - Chrome需要67+版本
   - 在地址栏输入：`chrome://version/` 查看版本

2. **清除浏览器数据**：
   - 设置 → 隐私设置和安全性 → 清除浏览数据
   - 选择"缓存的图片和文件"

3. **使用无痕模式**：
   - 打开无痕窗口
   - 重新访问网站

4. **检查网络**：
   - 确保网络连接正常
   - 尝试使用移动数据

### 问题2：安装失败

**解决方案**：
1. **重新加载页面**：
   - 下拉刷新页面
   - 等待完全加载

2. **检查存储空间**：
   - 确保手机有足够存储空间

3. **重启浏览器**：
   - 完全关闭Chrome
   - 重新打开

### 问题3：PWA不工作

**解决方案**：
1. **检查HTTPS**：
   - 确保使用HTTPS访问
   - 不要使用HTTP

2. **检查Service Worker**：
   - 在地址栏输入：`chrome://serviceworker-internals/`
   - 查看是否有错误
