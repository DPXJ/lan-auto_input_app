🚀 自动输入工具 - 启动说明

注意：由于浏览器安全限制，剪贴板功能需要在HTTP服务器环境下运行，直接打开HTML文件可能无法正常工作。

方法一：使用Python启动本地服务器

如果你安装了Python，可以在项目目录下运行以下命令：

# Python 3.x python -m http.server 8000 # Python 2.x python -m SimpleHTTPServer 8000

然后在浏览器中访问：http://localhost:8000

方法二：使用Node.js启动服务器

如果你安装了Node.js，可以使用以下命令：

# 安装http-server npm install -g http-server # 启动服务器 http-server -p 8000

然后在浏览器中访问：http://localhost:8000

方法三：使用VS Code Live Server

如果你使用VS Code编辑器：

  1. 安装"Live Server"扩展
  2. 右键点击index.html文件
  3. 选择"Open with Live Server"

方法四：直接访问（部分功能受限）

如果以上方法都不适用，你也可以直接打开HTML文件，但剪贴板功能可能无法正常工作：

直接打开应用
推荐：建议使用Python或Node.js启动本地服务器，这样可以确保所有功能正常工作。

功能测试

启动成功后，你应该能够：

  • 看到自动聚焦的输入框
  • 输入内容后点击发送或按回车
  • 内容自动复制到剪贴板
  • 在历史记录中看到卡片式展示
  • 点击历史记录卡片上的复制按钮

移动端测试

为了获得最佳体验，建议在移动设备上测试：

  1. 确保电脑和手机在同一网络
  2. 找到电脑的IP地址（如192.168.1.100）
  3. 在手机浏览器中访问：http://192.168.1.100:8000
